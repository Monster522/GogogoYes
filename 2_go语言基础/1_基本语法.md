## 基础语法

### 1.Go语言结构

```go
// 包声明-每个go文件必须属于某一个包
package main
// 引入依赖(包)
import "fmt"

// 函数-包括变量,语句,表达式,注释
func main() {
   fmt.Println("Hello, World!")
}
```

### 2.语句组成

- **行分隔符**

  ```go
  fmt.Println("Hello, World!")
  fmt.Println("the manager!")
  ```

  1. 一行代表一个语句结束。每个语句不需要像其它语言一样以分号 ; 结尾，这些工作都将由 Go 编译器自动完成。
  2. 将多个语句写在同一行，则必须使用 ; 人为区分，但是一般不这样做。

- **注释**

  1. 注释不会被编译，每一个包应该有相关注释。

  ```go
  // 单行注释
  /*
   多行注释
   第二行注释
   */
  ```
